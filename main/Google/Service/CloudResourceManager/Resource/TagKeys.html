TagKeys

class TagKeys extends Resource (View source)

The "tagKeys" collection of methods.

Typical usage is:

 $cloudresourcemanagerService = new Google\Service\CloudResourceManager(...);
 $tagKeys = $cloudresourcemanagerService->tagKeys;
